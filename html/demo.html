Front vs backend

Frontend: The client-side part of an application that focus on presenting stuff to the end-user

Backend: The server-side part of an appliation that focus on the behind-the-scenes, logic, computations, and data storage.

3 basic web technologies:

  • HTML: Markup language that describes the layout and content of a static web page.
  • javascript:Client side scripting language used to add dynamic functionality to a webpage. DOM object för HTML fil kan den komma åt och ändra sen
  • CSS: A language used to style a webpage. Beskriver hur kategorier av element ska se ut, t.ex. alla h2 taggar ska ha viss färg, font osv
